# SSFormCells
Collection of custom UITableViewCells.

## Requirements
Cocoapods
Swift
XCode

## Installation

SSFormCells is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SSFormCells'
```
